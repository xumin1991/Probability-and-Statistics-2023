概率论与数理统计 (经) 2023 年秋
===============

+ 主讲人：许岷 (xm@bjut.edu.cn)
+ 上课时间：
+ 上课地点：

## 1. 课程目标

### 概率论:
+ 概率的定义及性质
+ 一元随机变量及其分布、常见离散、连续随机变量的分布、随机变量变换
+ 二元随机变量及其联合分布、二元随机变量变换
+ 数字特征：均值、方差、协方差
+ 大数定律、中心极限定理

### 数理统计:
+ 总体与样本、统计量及其分布
+ 点估计：矩估计、极大似然估计、估计的评价准则
+ 区间估计
+ 假设检验
